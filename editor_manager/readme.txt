Editor Manager

＝＝＝＝ 準備 ＝＝＝＝

ダヴィンチリゾルブをインストールする。
Python をインストールする。
コマンドプロンプトで pip install youtube_dl を実行する。
このフォルダ内に api.key ファイルを（無ければ）作る（メモ帳で新しいファイルを保存し、名前を変えるなどする）。
api.key ファイルには、API のアクセス先を書き込む。

＝＝＝＝ 注意 ＝＝＝＝

このフォルダのパスに日本語を含まないこと。
このフォルダを動かさないこと（生成されるスクリプトに絶対パスが含まれるため）。

＝＝＝＝ 使い方 ＝＝＝＝

run.bat を実行する。
データベースで管理している番号、配信者の名前、動画のタイトル、コラボしたと思われる人物の名前、開始日時、動画の長さ、「↓」ボタンがある。
編集したい動画の「↓」ボタンをクリックする。
少し待つ。
コマンドプロンプトに「完了 （フォルダ名） （動画のタイトル）」が表示される。
コマンドプロンプトもしくは Editor Manager のウィンドウを閉じてもよい。
フォルダに移動すると、動画ファイルと resolver.lua ファイルがある。

ダヴィンチリゾルブを開く。
新しいプロジェクトを作る（名前は何でもよい）。
メニューのワークスペースからコンソールを開く。
resolver.lua ファイルをコンソールにドラッグアンドドロップする。
コンソールに動画の解像度とフレームレートが出力される。
コンソールを閉じる。

ダヴィンチリゾルブの画面右下の歯車アイコンをクリックして、プロジェクト設定画面を開く。
「マスター設定」の「ビデオモニタリング」の「ビデオフォーマット」を、コンソールに表示されていた解像度とフレームレートに設定する。
設定を保存する。

動画編集してください。

＝＝＝＝ 設定 ＝＝＝＝

config.ini ファイルに設定が保存されています。
config.ini ファイルを変更して保存した場合、Editor Manager を再起動する必要があります。

config.ini の中身 >>>
[DEFAULT]
# 切り抜きの前後を増やす（秒数指定）
clip_extend = 60
