Editor Manager

準備

ダヴィンチリゾルブをインストールする。
Python をインストールする。
コマンドプロンプトで pip install youtube_dl を実行する。
api.key ファイルを作る。

注意

このフォルダのパスに日本語を含まないこと。
このフォルダを動かさないこと（生成されるスクリプトに絶対パスが含まれるので）。

使い方

run.bat を実行する。
左に番号、真ん中に動画のタイトル、右に「↓」ボタンがある。
編集したい動画の「↓」ボタンをクリックする。
少し待つ。
コマンドプロンプトに「完了 （フォルダ名） （動画のタイトル）」が表示される。
コマンドプロンプトもしくは Editor Manager のウィンドウを閉じてもよい。
フォルダに移動すると、動画ファイルと resolver.lua ファイルがある。

ダヴィンチリゾルブを開く。
新しいプロジェクトを作る（名前は何でもよい）。
メニューのワークスペースからコンソールを開く。
resolver.lua ファイルをドラッグアンドドロップする。
コンソールを閉じる。

動画編集してください。
